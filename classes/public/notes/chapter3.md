# Chapter 3: Ethics

## types of ethics
- Ethical absolutism: behave the same in all situations, even if doing so hurts loved ones.
- situation ethics: A shifting code that varies depending on situation, usually under extenuating circumstances
- culturally relative: ethics can vary across societies.
    - some cultures may not understand "raining cats and dogs"
    - use proper citations
- first amendment: freedom of speech.
    - although you have freedom of speak, you still need to be ethical.
- false inference leads to incorrect conclusion


## citing work
